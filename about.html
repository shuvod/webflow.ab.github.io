About


Our Mission

To empower designers, creative
professionals, and entrepreneurs to create
for the web.

We Believe

The web changes everything

Never before have so many people had so much access to such a
powerful medium for building businesses; sharing knowledge,
insight, and creativity; and creating communities.

Everyone should have the
power to create for the web

To make the most of the web, we need tools that democratize
access to it i.e., tools that empower creative people to build
websites and share their visions with the world.

Platforms should enable
access without sacrificing
flexibility

Instead of locking you into customizable templates or forcing
you to learn code, web publishing platforms should empower you
to build whatever you like.

Platforms should encourage
collaboration

A web design platform should unite code, content, and design,
helping people in each discipline collaborate, and letting others
experience all three working in concert.
