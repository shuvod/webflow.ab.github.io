Webflow Features


Web Design


Add CSS filters to anything

Create beautiful effects by adding one or more of 8 CSS filters to any element of your Webflow websites
inside the Designer.

Add any code you need

Need to do something Webflow doesn't support out of the box? You can easily add custom code for the entire site, a specific page,
or a specific part of a page.

Clean, compliant, exportable code

Webflow writes clean, W3C compliant, semantic code for you. Without all the messy inline classes you've come to expect
from drag-and-drop site builders. You can even export your HTML, CSS, and images in a handy ZIP file you can use however you like.

Every CSS property

Enjoy total design freedom with access to many CSS styling properties from the latest specs, from alpha to Z-index.

Motion design

3D transforms and animations

Add a whole new dimension to your website designs with the first tool that lets you create 3D CSS transforms-without writing code.

Design interactions without code

Easily add complex interactions and animations to your site, without even looking at JavaScript.
